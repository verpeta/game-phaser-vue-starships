# Game: starships

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

## Built-on
- [Phaser3](https://github.com/photonstorm/phaser)
- Vue.js - v2+
- socket.io
- node.js (as backend server) - v14+
- mongoDb 
- docker + docker-compose
